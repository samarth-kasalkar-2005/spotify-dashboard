# 🎵 Spotify Analytics Dashboard

A premium Spotify-inspired analytics dashboard built using Streamlit, Plotly, and Python.  
This dashboard provides interactive music insights, artist popularity analysis, genre trends, and audio feature visualizations with a modern Spotify-style UI.

---

## 🚀 Features

- 🎧 Spotify-inspired dark green UI
- 📊 Interactive analytics dashboard
- 🎤 Top artists analysis
- 🎵 Genre-based filtering
- 📈 Popularity distribution charts
- 🌊 Audio feature analysis
- ⚡ Energy vs Danceability visualization
- 🔥 KPI metric cards
- 🎨 Modern glassmorphism styling
- ⏳ Animated loading screen
- 📱 Responsive dashboard layout

---

## ▶️ Run Locally

Clone the repository:

```bash
git clone <your-repository-link>
```

Go to project folder:

```bash
cd Spotify-Dashboard
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

### Windows
```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run app.py
```
